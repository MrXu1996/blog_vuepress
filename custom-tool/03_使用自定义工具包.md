## 	3.1 下载工具包

```shell
# 名称是你前面指定的库的名称
npm install a-utils2
```

## 	3.2 网页中引入并使用

```html
<script src="./node_modules/a-utils2/dist/atguigu-utils.js"></script>
<script>
  aUtils.test()
</script>
```

## 	3.3 模块化引入并使用

```html
<script src="./node_modules/a-utils2/dist/atguigu-utils.js"></script>
<script>
  aUtils.test()
</script>
```
